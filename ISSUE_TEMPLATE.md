# Issue Template untuk GitHub Achievements

## Achievement yang Bisa Didapatkan:

### 1. Starstruck 🌟

- **Aksi**: Star repository ini
- **Cara**: Klik tombol ⭐ di halaman repository

### 2. Issuer 🐛

- **Aksi**: Buat issue baru
- **Cara**:
  1. Klik tab "Issues"
  2. Klik "New issue"
  3. Pilih template ini atau buat issue manual

### 3. Forker 🔄

- **Aksi**: Fork repository lain
- **Cara**: Cari repository menarik dan klik "Fork"

### 4. Pull Shark 📝

- **Aksi**: Buat Pull Request yang di-merge
- **Cara**: Contribute ke repository open source

## Contoh Issue untuk Achievement:

### Feature Request

**Title**: Add new image filters
**Body**:

```
I would like to suggest adding new image filters to enhance the editor:

- Vintage filter
- Black and white with custom intensity
- Color temperature adjustment
- Sharpness enhancement
- Noise reduction

This would make the image editor more comprehensive and professional.
```

### Bug Report

**Title**: Minor UI improvement needed
**Body**:

```
I noticed a small UI issue:

**Description**: The filter sliders could be more responsive on mobile devices
**Expected**: Smooth sliding experience
**Actual**: Sometimes laggy on touch devices

This is a minor enhancement that would improve user experience.
```

### Enhancement

**Title**: Add keyboard shortcuts
**Body**:

```
It would be great to add keyboard shortcuts for common actions:

- Ctrl+Z: Undo
- Ctrl+Y: Redo
- Ctrl+S: Save
- Ctrl+O: Open file
- Ctrl+R: Reset image

This would make the editor more efficient for power users.
```
